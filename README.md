\# Torre de Bloques - Pila (LIFO) en Python



\## Descripción

Este proyecto muestra cómo funciona una \*\*pila (LIFO - Last In, First Out)\*\* 

usando \*\*Programación Orientada a Objetos (POO)\*\* en Python.



Se simula un robot que construye una torre de bloques, donde \*\*el último bloque agregado\*\* 

es \*\*el primero que se retira\*\*.  



Esto permite ver claramente el comportamiento de una pila en un escenario práctico.



---



\## Funcionalidades

\- Agregar bloques a la torre

\- Retirar el bloque superior

\- Ver el bloque superior sin retirarlo

\- Desmontar toda la torre en orden LIFO

\- Llevar un historial de los bloques retirados



---



\## Conceptos Utilizados

\- Programación Orientada a Objetos (Clases y Métodos)

\- Estructura de datos \*\*Stack (LIFO)\*\*

\- Encapsulación y abstracción

\- Operaciones O(1) para agregar o quitar bloques
